LibProgressMonitor

概要

使用感は SWT の ProgressMonitor と似ています。似てはいますが、起動方法が異なります。

参考までに対応表です:

Eclips
Class ProgressMonitorDialog WaitForm
Interface IProgressMonitor IProgressMonitor
Interface IRunnableWithProgress IRunnableWithProgress
Class NullProgressMonitor NullProgressMonitor
なし ProgressMonitorDelegateProxy
Class ProgressMonitorWrapper ProgressMonitorWrapper
Class SubProgressMonitor SubProgressMonitor

起動例

try
{
using (var form = new WaitForm())
{
form.run(
fork: true,
cancelable: true,
runnable: new Runnable()
);
}
}
catch (Exception ex)
{
MessageBox.Show(ex.ToString());
}
